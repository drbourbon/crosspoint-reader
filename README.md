Minor fork of main [CrossPoint Reader](https://github.com/crosspoint-reader/crosspoint-reader) that:

* Replaces Bookerly font (I don't like it) with [Lora](https://fonts.google.com/specimen/Lora)
* Sets bigger font sizes
* Adds support for [TRMNL BYOS](https://docs.trmnl.com/go/diy/byos) to fetch a TRMNL screen when going to sleep
